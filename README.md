## Описание варианта  
Вариант 3. Задача: Конвертер из десятичной системы счисления в двоичную.

## Описание реализации  
**Используемый класс:** Stack    
Класс импортируется из модуля my_stack и реализует структуру данных Стек (LIFO — Last In, First Out).  
Функция decimal_to_binary(decimal_number) преобразует целое десятичное число (включая отрицательные) в строку с двоичным представлением.  
**Сложность** Stack() - O(1); push(item) - O(1); pop() - O(1); is_empty() - O(1)
